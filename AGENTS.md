# Project Instructions

- Final project name: Immersive Deposit Scanner.
- Mod id: `immersive_deposit_scanner`.
- Java package root: `com.oblixorprime.immersivedepositscanner`.
- Target only Minecraft 1.21.1, NeoForge 21.1.x, and Java 21.
- JourneyMap, Immersive Engineering, and Immersive Petroleum are required dependencies.
- Keep JourneyMap client API imports under `client/journeymap`.
- Do not scan world blocks for deposits. Only record data exposed by valid core samples or survey result data.
- For every runtime build, use `install-mod.ps1` or equivalent verification to build, install to the LAB mods folder, check SHA-256, and confirm exactly one jar for this mod remains.
