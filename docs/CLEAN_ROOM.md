# Clean-Room Statement

Immersive Deposit Scanner is an original implementation of the public idea of tracking deposits revealed by legitimate core samples. It does not decompile, copy, translate, or derive code or assets from Mineral Tracker.

The implementation uses public runtime APIs and data exposed by:

- NeoForge
- JourneyMap API
- Immersive Engineering
- Immersive Petroleum

The mod never scans world blocks, searches hidden ores, performs x-ray behavior, or reveals information not already present in the sample or survey data being handled.
